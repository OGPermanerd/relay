# Employee Offboarding Checklist

Ensure smooth, compliant, and respectful employee departures with a comprehensive offboarding process.

## Purpose
This checklist ensures that when an employee leaves (voluntary or involuntary), all access is revoked, knowledge is transferred, equipment is returned, and the transition is handled professionally.

## Inputs
- **Employee Name**: {{name}}
- **Department**: {{department}}
- **Manager**: {{manager}}
- **Last Day**: {{last_day}}
- **Departure Type**: {{type}} (voluntary resignation / involuntary / layoff / retirement / contract end)
- **Notice Period**: {{notice}} (2 weeks / immediate / other)
- **Replacement Hired**: {{replacement}} (yes / no / internal transfer)

## Pre-Departure (Manager + HR)

### Immediate (within 24 hours of notice)
- [ ] Confirm last working day with employee
- [ ] Notify IT to schedule account deactivation for last day EOD
- [ ] Notify payroll of termination date
- [ ] Review any non-compete, NDA, or IP assignment obligations
- [ ] Determine if garden leave or immediate departure
- [ ] Plan team communication (with employee's input where appropriate)

### Knowledge Transfer (during notice period)
- [ ] Identify critical knowledge areas and document holders
- [ ] Schedule knowledge transfer sessions with replacement or team
- [ ] Document ongoing projects, status, and next steps
- [ ] Transfer ownership of shared documents and files
- [ ] Update project management tools (reassign tasks)
- [ ] Document vendor/client relationships and key contacts
- [ ] Record passwords for shared accounts in password manager (never personal)
- [ ] Transition recurring meetings and calendar invites

### Team Communication
- [ ] Manager communicates departure to immediate team
- [ ] Broader org notification (if appropriate)
- [ ] Client/vendor notification plan (who tells which external contacts)
- [ ] Update org chart and team page

## Last Day Checklist

### IT & Access (IT Team)
- [ ] Disable email account (set auto-reply with redirect)
- [ ] Revoke SSO / identity provider access
- [ ] Deactivate VPN access
- [ ] Remove from Slack, Teams, and communication tools
- [ ] Revoke access to code repositories (GitHub, GitLab)
- [ ] Remove from cloud services (AWS, GCP, Azure)
- [ ] Deactivate SaaS tool accounts (Salesforce, Jira, Figma, etc.)
- [ ] Remove from building access / badge system
- [ ] Revoke API keys and service account credentials
- [ ] Transfer ownership of shared Google Drive / OneDrive files
- [ ] Remove from distribution lists and shared mailboxes
- [ ] Wipe company data from personal devices (if BYOD)

### Equipment (IT Team)
- [ ] Laptop returned and wiped
- [ ] Monitor, keyboard, mouse returned (if issued)
- [ ] Mobile phone returned (if company-issued)
- [ ] Security badge / keys returned
- [ ] Parking pass returned
- [ ] Home office equipment returned (if provided for remote work)

### HR & Finance
- [ ] Process final paycheck (include accrued PTO payout per policy)
- [ ] COBRA / benefits continuation notice sent
- [ ] 401(k) / pension rollover information provided
- [ ] Stock option exercise window communicated (typically 90 days post-departure)
- [ ] Severance agreement signed (if applicable)
- [ ] Exit interview conducted
- [ ] References policy communicated
- [ ] Return of company credit cards
- [ ] Outstanding expense reports submitted and processed

### Legal & Compliance
- [ ] Remind employee of ongoing NDA obligations
- [ ] Confirm return of all confidential materials
- [ ] Document any IP created during employment
- [ ] Non-compete obligations reviewed and communicated
- [ ] Separation agreement executed (if applicable)

## Post-Departure (within 1 week)

- [ ] Verify all system access has been revoked (IT audit)
- [ ] Update emergency contact lists
- [ ] Archive employee email (per retention policy)
- [ ] Remove from payroll system
- [ ] Update benefits enrollment
- [ ] Close out any open IT tickets
- [ ] Send thank-you note or farewell message (if appropriate)
- [ ] Add to alumni network (if applicable)
- [ ] Review and redistribute workload if replacement not yet hired

## Departure Type Notes

### Voluntary Resignation
- Conduct genuine exit interview â€” this is valuable feedback
- Maintain positive relationship for potential rehire or referrals
- Offer to serve as reference (if appropriate)

### Involuntary Termination
- Coordinate closely with legal counsel
- Have separation agreement prepared before the conversation
- Immediate access revocation (before or during the conversation)
- Provide outplacement services if available

### Layoff / Reduction in Force
- Follow WARN Act notification requirements (if applicable)
- Prepare severance packages per company policy
- Provide job search support and references
- Communicate transparently with remaining team about impact

## Metrics to Track
- Average offboarding completion time
- Access revocation time (target: <4 hours after last day)
- Equipment return rate
- Exit interview participation rate
- Rehire rate (indicator of positive departures)
