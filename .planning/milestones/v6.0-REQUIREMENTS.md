## v6.0 Requirements

### Company IP Dashboard

- [x] **IPDASH-01**: Admin can view total skills captured, total uses, total hours saved, and active contributors on a company IP dashboard
- [x] **IPDASH-02**: Admin can see IP concentration risk — which employees hold the most critical skills (single-author skills with high usage)
- [x] **IPDASH-03**: Admin can see estimated replacement cost for each high-value skill based on usage, hours saved, and complexity
- [x] **IPDASH-04**: Admin can view org-wide quality trends over time (quality scores, feedback sentiment, benchmark results as trend charts)
- [x] **IPDASH-05**: Admin can see key person dependency alerts when critical skills have only one author
- [x] **IPDASH-06**: Admin can export an IP Report (PDF/CSV) summarizing IP captured, risk assessment, and estimated value for board presentations

### Individual Skills Portfolio

- [x] **PORT-01**: User can view their personal skills portfolio showing skills authored, total usage, total hours saved, and contribution ranking
- [x] **PORT-02**: User can see a portable vs company IP breakdown — which skills are personal-scoped (theirs forever) vs tenant-scoped (company's)
- [x] **PORT-03**: User can view a skills impact timeline showing when they created/improved skills and cumulative impact over time
- [x] **PORT-04**: User can generate a "Skills Resume" — a shareable, formatted summary of their skills and impact for use in job applications
- [x] **PORT-05**: User can see an impact calculator showing how much value they have added to the company over time through their contributions to the skillset
- [x] **PORT-06**: User can load pre-LLM history (emails, documents produced, prior work artifacts) to demonstrate historical impact and inform skill recommendations
